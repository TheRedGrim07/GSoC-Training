Core Concept: The Server (Waiter) waits for a Request (Order) at a specific Route (Table) and returns a Response (Food).

1. The Server (server.py)

from flask import Flask, request

app = Flask(__name__)

# Pattern 1: The Basic Route (GET)
# URL: http://localhost:5000/
@app.route("/")
def home():
    # ALWAYS return a Dictionary (JSON), never plain text
    return {"status": "online", "system": "GSoC Node 1"}

# Pattern 2: The Path Variable (GET)
# URL: http://localhost:5000/user/Ayush
# Use this for specific resources (IDs, names)
@app.route("/user/<name>")
def get_user(name):
    # 'name' is passed as an argument to the function
    return {
        "user": name, 
        "length": len(name) # Python's length function
    }

# Pattern 3: The Query Parameter (GET)
# URL: http://localhost:5000/search?q=python&page=1
# Use this for filters, search, or optional settings
@app.route("/search")
def search():
    # request.args.get(key, default_value, type)
    # ALWAYS provide a default value to prevent crashes!
    query = request.args.get('q', "nothing") 
    page = request.args.get('page', 1, type=int) 
    
    return {"searching_for": query, "page_number": page}

# Pattern 4: The Secure Envelope (POST)
# URL: http://localhost:5000/login
# Use this for sending private data (passwords, large text)
@app.route("/login", methods=['POST'])
def login():
    # 1. Open the Envelope (Body)
    data = request.get_json()
    
    # 2. Safety Check (Safe Navigation)
    if not data:
        return {"error": "No data sent"}, 400
        
    # 3. Process
    username = data.get('username')
    return {"status": "received", "user": username}

if __name__ == "__main__":
    app.run(debug=True) # debug=True helps you see errors

2. The Client (client.py)

Used to test POST requests since Chrome can't do it easily.

import requests

# The Payload (The Letter inside the Envelope)
payload = {"username": "admin", "password": "123"}

# The Handshake
# json=payload automatically formats it nicely
response = requests.post("http://127.0.0.1:5000/login", json=payload)

print(response.json())

‚ö†Ô∏è The Engineer's Rules

  1. Never Return Strings: Always return a Dictionary {}. APIs speak JSON.
  2. Never Trust Input: Use request.args.get('key', default) to handle missing data.
  3. GET vs POST:
    • GET: Data is in the URL. (Visible, Unsafe, Small).
    • POST: Data is in the Body. (Hidden, Secure, Large).
